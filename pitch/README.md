# Voodoo Hut Pitch Presentation

An interactive, self-contained HTML presentation for pitching the AV & Broadcast Integration project.

## Getting Started

Open `index.html` in any modern web browser. No dependencies, no network required.

**Navigation:**
- **Spacebar / Right Arrow:** Next slide
- **Left Arrow:** Previous slide  
- **Click anywhere:** Next slide
- **Dot navigation:** Click dots on the right to jump to a slide

## Adding Logo Images

To add actual logo images to the presentation, save these files to the `assets/` directory:

- `assets/voodoo-hut-logo.png` — Main Voodoo Hut logo
- `assets/htxpunk-logo.png` — HTXPunk production company logo

The presentation currently uses text-based branding. You can optionally replace these with actual image files by modifying the HTML (search for the HTXPunk and venue name sections).
