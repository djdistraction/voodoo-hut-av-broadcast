Each broadcast bus has mastered EQ, compression, and limiting to –16 LUFS before leaving the console. This is not a FOH mix. This is a broadcast mix.

HOW WE SET IT UP

Configuration & Automation

The "Set It Once" Philosophy

After Phase 8 is complete, the system runs on automation. Live show starts → BUTT connects → Liquidsoap crossfades house music to live feed → Back to playlist at show end. Zero manual transitions.

BROADCAST MIX CONFIGURATION

Bus 9/10 (Main Stage): HPF 50Hz | –2.5dB@300Hz | +1.5dB@6kHz | Precision Limiter → –16 LUFS Bus 11/12 (Corner Stage): HPF 50Hz | –3.0dB@300Hz | +2.0dB@8kHz | Precision Limiter → –16 LUFS Bus 13/14 (Patio Stage): HPF 50Hz | –2.0dB@250Hz | +1.8dB@7kHz | Precision Limiter → –16 LUFS

Each stream is independently mastered. Each stage sounds professional without any manual tweaking night-to-night.
